# 配电网-道路协同抢修：实现规划（基于会议纪要与前期报告）

## 1. 规划依据
本规划综合以下两份材料：
- `会议纪要_配电网道路协同抢修建模.md`
- `交通-电力协同抢修策略模型与算法前期报告v2.pdf`

结论一致：应先做“可解释、可运行、可复现”的MVP，优先完成双层图建模、基础策略仿真、恢复曲线输出。

## 2. 分阶段实现路线

### 阶段 A（MVP，1~3 天）
1. 双层网络数据结构（道路层、电网层、跨层映射）。
2. 基础可达性与供电恢复判定。
3. 两类策略骨架：
   - S1：先修路后修电
   - S2：路电并行
4. 输出主指标曲线：`LSD(t)`（恢复供电负荷/线路长度）。

### 阶段 B（增强版，3~7 天）
1. 加入关键设施（医院/安置点/应急中心）恢复时效约束。
2. 加入昼夜效率折减与停电导致的效率影响。
3. 初步加入“机动投送”抽象能力（不细化飞行航线）。

### 阶段 C（研究版，7 天+）
1. 启发式优化（优先级评分、桥边识别、关键割集）。
2. 多智能体/学习方法作为增量模块对比。
3. 论文图表自动化输出（曲线、甘特图、策略对比表）。
